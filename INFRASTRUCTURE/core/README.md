# core

**Core Utilities & Helpers**

This folder contains shared utilities that support all pipeline stages.

## Contents

- `config.js` — Configuration management
- `log.js` — Centralized logging
- `store.js` — Data storage interface
- `errors.js` — Error handling utilities
- `taskRegistry.js` — Task tracking
- `health.js` — Health monitoring
- `migrations.js` — Database migrations
- Other shared helpers

## Principle

Core utilities are stable. Changes require careful consideration of impact across all departments.
