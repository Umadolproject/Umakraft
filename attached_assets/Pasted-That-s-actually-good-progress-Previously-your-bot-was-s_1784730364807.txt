### Where to look

Since the crash happens **after**:

```
Registered event: interactionCreate
```

the next thing your program loads is likely one of:

* a command file
* a utility module
* a database module
* a handler imported after the events

### Common causes

Look for things like:

```js
const * = something;      // ❌
```

```js
import { * } from "...";  // ❌
```

```js
export * something;       // ❌
```

or even Markdown accidentally copied into a `.js` file:

```text
***
```

or

```markdown
# README
```

These will all produce parser errors before your code can run. ([MDN Web Docs][1])
