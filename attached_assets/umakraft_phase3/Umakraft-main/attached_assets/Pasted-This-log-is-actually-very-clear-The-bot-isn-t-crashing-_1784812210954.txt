## What to check

### 1. Railway Variables

Make sure you have **exactly** this variable:

```
DISCORD_TOKEN
```

Not:

```
discord_token
BOT_TOKEN
TOKEN
DiscordToken
```

Environment variable names are **case-sensitive**.

### 2. Does Railway have the value?

```

Not:

### 3. Is it in the correct service?

If your Railway project has multiple services:

```

Redis
Database

Putting it on the Hugging Face service won't make it available to the bot.
